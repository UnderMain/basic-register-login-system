Installieren:

1. Die Website auf euren Server hochladen! (Anforderungen: min PHP 5.3 und Web-Server)
2. In der Config alle Variablen anpassen z.b. Server Name oder Sever Domain u.a. Datenbank
3. Datenbank eintragen in die Config (Siehe Schritt Datenbank(1-4))
4. Auf die install Website gehen:
	- deinedomain.de/ticket-system/install
5. Dort könnt ihr den Admin Account erstellen! 
6. !!!Ganz wichtig nachdem ihr euren Account erstellt habt sofort install Ordner löschen!!!
7. Mit dem Admin Account könnt ihr weiter Accounts erstellen für z.b. Supporter
8. Das wäre es auch mit der Website!


Datenbank:
1. In PHPmyAdmin einen neuen Benutzer erstellen und ihm eine Datenbank und alle Rechte geben.
2. Auf die Datenbank klicken und dann auf importieren.
3. Dann importierest du ticket.sql und user.sql in die Datenbank damit das System funktioniert.
4. Alles Ready!

Minecraft Server:
1. Plugin(Ticket-System) in den Plugin Ordner ziehen!
2. Server starten und stoppen.
3. MySQL Datei bearbeiten und bei belieben auch die config Datei
4. Server neu starten und es funktioniert.
5. Spieler können nun mit dem recht ticket.write ein ticket mit /ticket <email> <thema> <nachricht> erstellen.
6. Weitere Befehle folgen!
